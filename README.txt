All'interno della cartella compressa saranno disponibili 3 file, in particolare:

- Documentazione relativa al progetto in formato PDF;
- Il progetto in formato HTML per una rapida visualizzazione;
- Il progetto in formato .json (o anche .ipynb) per eseguire il codice;
- Immagine di una TAC in formato .png;

Per aprire il file in formato json all'interno di un notebook Jupyter, occorre avere
Anaconda (o Miniconda) installato, importare il suddetto file attraverso il pulsante 
'Upload' ed inserire all'interno della stessa cartella di lavoro il file .png.
